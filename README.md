FunFlow 是一个短视频浏览与分享应用，致力于为用户提供流畅、有趣的短视频消费体验。

## Features

- **视频推送**：用户兴趣推送、相似视频推荐、标签分类列表、热门视频排行榜
- **用户生态**：登录注册、信息管理、视频创作、互动功能（点赞、收藏、评论）
- **基础服务**：对象（图片、视频）存储、内容（文本、图片、视频）审核机制
